# 屋敷状況板

> お嬢様屋敷の進行状況を一覧する状況板。
> 家政婦が「お嬢様からの cmd 受領時 / メイド完了報告受領時 / お嬢様への完了報告時」に更新する。

最終更新: 2026-04-28

## 運用ルール

- **更新主体**: kaseifu（家政婦）。お嬢様・メイドは原則更新しない。
- **更新タイミング**:
  1. お嬢様から新規 cmd（指令）を受領したとき → 「進行中のお役目」に追記
  2. メイドから完了報告（`queue/maid_NN_report.yaml`）を受領したとき → 状態を更新、必要なら「完了したお役目」に移動
  3. 要相談事項が発生したとき → 「🚨要相談」に追記
  4. お嬢様へ最終報告したとき → 「完了したお役目」に確定、ストリーク更新
- **時刻形式**: `YYYY-MM-DD HH:MM`（24時間制、JST）
- **状態（status）の取り得る値**: `assigned` / `in_progress` / `blocked` / `review` / `done` / `failed`
